### 开发测试阶段的打包部署

- [ ] 配置登录远程机器的 SSH config
- [ ] 本地使用 Gradle 或者 Maven 打包
- [ ] 使用 scp 安全拷贝 jar 包到远程机器
- [ ] 使用 `sudo apt install openjdk-8-jre-headless` 安装 JDK 环境
- [ ] 使用 `java -jar` 完成部署

### 分布式打包部署

- [ ] 在 docker 中部署 etcd
- [ ] 部署 kafka
