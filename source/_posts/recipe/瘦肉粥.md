## 食材準備

| 材料 | 參考量              | 參考單價 | 參考價格 |
| ---- | ------------------- | -------- | -------- |
| 瘦肉 |                     | ?        | 2.2元    |
| 米   | 80ML（煮成約3碗粥） | 4元/斤   | <1元     |

## 烹飪過程

1. 淘米，根據具體電飯煲的要求加入適量的水，煮粥；
2. 將瘦肉切成細長狀肉絲，加鹽爪拌，醃製10分鐘；
3. 粥快熟了，加入預備好的瘦肉，攪拌，繼續煮10分鐘；
4. 開鍋享用。
