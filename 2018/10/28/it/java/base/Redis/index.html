服务器连接 Redis 失败

（1）背景

服务器部署 Redis ，代码部署在另一主机，调用 Redis 时发生以下异常：

（2）原因

查看 redis.conf 发现 bind 127.0.0.1 ，意味着只有部署 Redis 的机器 可以正常访问 Redis。

还有 Redis 运行在保护模式下。

（3）解决方案

方案一：配置 bind 地址为内网的 IP 地址，修改 protected-mode no ，然后重启 Redis。
