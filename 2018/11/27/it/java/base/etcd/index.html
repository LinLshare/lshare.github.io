    • etcd 是什么？（What is etcd）

    etcd 是一个一致的分布式可靠的键值存储技术。可被用来做配置共享和服务发现。

    • 开发语言：Go
    • 共识算法：Raft
    • 命名来源：表示分布式的 etc 目录，发音为/ˈɛtsiːdiː/。
    • 使用的端口：2.0 后，使用 2379 作为外部客户端通信，使用 2380 作为内部服务间通信。
    • 发起团队：CoreOS
